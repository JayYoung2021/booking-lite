# 酒店管理系统

## 需求分析

### 原文要求

- 客房标准制定，包括房间号，房间面积，床位数量，住房单价
- 客房标准信息修改查询
- 基本信息输入
- 剩余客房信息的查询
- 订房信息输入，包括客房编号，客房单价，顾客姓名，顾客身份证号码，顾客手机号，入住日期，备注
- 订房信息的修改，增加，查询
- 客房居住费用结算的实现
